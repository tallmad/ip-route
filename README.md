VPN auto routing scripts - for Mac OS X
==========================

1. Install

    ```shell
    sudo ./ip-route install
    ```

1. Edit configuration

    > more detail in /etc/ppp/ip-route.ini

1. Uninstall

    ```shell
    sudo ./ip-route uninstall
    ```

1. Fetch gfw segments

    ```shell
    sudo ./ip-route fetch_gfw_segment
    ```
